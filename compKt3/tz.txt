Техническое задание:
Создать пользовательский веб-компонент с использованием шаблонов (templates) и слотов (slots), 
который позволит пользователям создавать и отображать карточки с контентом по своему выбору.
Целью данного проекта является создание пользовательского веб-компонента, который будет использовать 
шаблоны и слоты для создания и отображения карточек с контентом.
 
Описание функциональности:
Компонент должен предоставлять шаблон карточки, включая заголовок и текстовое содержание.
Компонент должен предоставлять слоты для вставки пользовательского контента внутри карточки, как, 
например, изображение или дополнительный текст.
Пользователь должен иметь возможность создавать несколько карточек на одной странице с различным 
контентом.
 
Технические требования
Разработка компонента должна быть выполнена с использованием HTML, CSS и JavaScript.
Используйте шаблоны (templates) для определения структуры карточки, включая заголовок и текстовое 
содержание.
Используйте слоты (slots), чтобы разрешить вставку пользовательского контента внутри карточки.
Компонент должен быть стилизован с использованием CSS для обеспечения приятного внешнего вида.
